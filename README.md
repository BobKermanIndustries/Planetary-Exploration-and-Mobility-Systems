# Planetary Exploration and Mobility Systems v0.1.1
A mod for Kerbal Space Program 1.12.x by Bob Kerman Industries.

This mod adds a set of parts designed to emulate the Curiosity and Mars 2020 Perseverance Mars rovers with a working rocker-bogie suspension system and some of the EDL hardware. Reccomended to use with the Atlas V from Bluedog Design Bureau.
For how to assemble the rover, please consult the manual provided and in-game part descriptions.

This mod has several dependencies, without them installed, it will not function correctly, a hard requirement is the Breaking Ground DLC.

INSTALLATION:
Place the contents of GameData/ into your KSP/GameData/ directory.
Merge the Ships/ folder with the Ships/ folder inside your KSP_win64/ directory.

DEPENDENCIES
- Breaking Ground DLC
- KSP Community Fixes
- Deployable Engines
- Module Manager
- B9PartSwitch
- Hullcam VDS Continued
- DMagic Module Science Animate
- RealChute
- Custom Parachute Message
